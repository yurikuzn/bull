# Bullbone

A front-end library developed for EspoCRM. See [docs](https://docs.espocrm.com/development/view/).

## Features

* Nested views
* Layout engine
* Events (from Backbone.js)

## Building

Bundles: `npm run prepare`

Test specs: `npm run build-test`
